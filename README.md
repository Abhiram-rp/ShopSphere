# ShopSphere
ShopSphere is an ecommerce platform built using springboot 

- Microservices Architecture
    - The application is divided into independent User, Product, and Order services.
    - Each service has its own dedicated database and manages its own data.
    - Services can be developed, deployed, and scaled independently.
    - This architecture improves scalability, maintainability, and fault isolation.
